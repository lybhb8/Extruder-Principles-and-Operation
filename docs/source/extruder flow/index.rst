=========================
6 单螺杆挤出机熔体流动机理
=========================

本章通过简化分析，推导出几种典型螺杆类型熔体输送段的输出量、压力与螺杆转速之间的关联方程。文中探讨了螺杆内部的流动模式及沿螺杆轴向的压力分布，并分析了泄漏流现象。通过图形化展示分析了螺杆与模头组合在简单及复杂工况（含排气）下的输出/压力特性。讨论了对简化方程的修正项，包括因假塑性流动和非等温条件导致的修正。固体输送与熔融机制将在第七章详述。
